# hennge-challenge
My application to Hennge's front-end software engineer position

### To Run
1. If not already, install Node.js - https://nodejs.org/

2. `cd mail-archiver`

3. `npm i`

3. `ng serve`

4. Navigate in your browser to localhost:4200
